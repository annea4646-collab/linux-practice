🚀 Linux & Python Practice Box
파이썬 기초 수업의 핵심 내용을 Linux(Ubuntu/WSL) 환경에서 직접 구현하고 테스트한 기록입니다. 
실제 서버 환경에서의 데이터 처리와 시스템 작동 원리를 이해하는 것을 목표로 합니다.

📜 파이썬 학습 내용 요약 (260112 ~ 260120)
1. 260112_practice.py: 파이썬 코딩 스타일과 기초
PEP 8 가이드라인: 들여쓰기, 줄 길이 제한(79자), 클래스/함수 간 빈 줄 사용 등 파이썬 표준 스타일을 학습했습니다.

명명 규칙: 변수/함수에는 snake_case, 클래스에는 CamelCase, 상수에는 UPPER_CASE를 사용하는 관습을 익혔습니다.

주석과 도큐스트링: 코드 설명을 위한 # 주석과 함수/클래스 정보를 저장하는 docstring('''...''')의 차이를 확인했습니다.

접근 제한자: 파이썬에는 강제적인 접근 제한자가 없지만, _name(protected)이나 __name(private)과 같은 명명 관습을 통해 접근 범위를 제어함을 실습했습니다.

2. 260113_practice.py: 클래스 심화 및 리소스 관리
인스턴스 변수와 메모리: self를 사용하여 인스턴스 변수를 생성하고, 객체별로 독립적인 메모리 주소(id)를 가짐을 확인했습니다.

생성자 (__init__): 객체 생성 시 데이터를 강제로 주입하거나 초기화하는 방법을 학습했습니다.

클래스 변수: 모든 객체가 공유하는 변수의 특징과 사용법을 익혔습니다.

컨텍스트 매니저: with open(...) 구문을 통해 파일 열기/닫기를 안전하게 관리하는 방식을 실습했습니다.

3. 260115_practice.py: 상속과 모듈화
상속 및 오버라이딩: 부모 클래스의 기능을 물려받거나(pass), 필요한 부분만 재정의(SafeFourCal)하는 방법을 학습했습니다.

다중 상속 및 super(): 여러 클래스를 상속받을 때 super()를 사용하여 부모 클래스의 메서드를 호출하는 순서(MRO)를 확인했습니다.

추상 클래스 (abc 모듈): abstractmethod를 통해 하위 클래스에서 반드시 구현해야 하는 메서드 틀을 만드는 법을 익혔습니다.

모듈 실행 조건: if __name__ == "__main__":을 사용하여 파일 직접 실행 시에만 특정 코드가 작동하도록 제어하는 기법을 실습했습니다.

4. 260119_practice.py: 예외 처리와 내장 함수
사용자 정의 예외: Exception 클래스를 상속받아 나만의 에러 클래스(MyError)를 만들고 raise로 발생시키는 법을 학습했습니다.

내장 함수 활용: abs, all, any, enumerate, filter, map, sorted 등 파이썬에서 자주 쓰이는 핵심 함수들을 실습했습니다.

데이터 타입 변환: 문자열(str)과 바이트(bytes) 간의 인코딩(utf-8) 및 디코딩 과정을 확인했습니다.

5. 260120_practice.py: 표준/외부 라이브러리 및 시스템 제어
파일 시스템 패턴 매칭: glob 모듈을 사용하여 특정 패턴의 파일 목록을 읽어오는 방법을 실습했습니다.

객체 직렬화: pickle 모듈로 파이썬 객체를 파일로 저장하고 불러오는 과정을 학습했습니다.

환경 변수 및 시스템: os.environ을 통해 시스템 환경 변수를 확인하고 제어하는 법을 익혔습니다.

JSON 데이터 처리: json.dumps와 json.loads를 사용하여 파이썬 딕셔너리와 JSON 문자열 간의 상호 변환을 실습했습니다.
